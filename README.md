# SKlauncher Documentation

This website is built using [VitePress](https://vitepress.dev/), a static site generator powered by Vite and Vue.

### Installation

```
$ pnpm install
```

### Local Development

```
$ pnpm dev
```

This command starts a local development server and opens up a browser window. Most changes are reflected live without having to restart the server.

### Build

```
$ pnpm build
```

This command generates static content into the `.vitepress/dist` directory and can be served using any static contents hosting service.

### Preview

```
$ pnpm preview
```

Serves the production build locally for verification.

### Contributing

Feel free to contribute if you find any mistakes or want to add anything to the documentation!
