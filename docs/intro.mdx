# Introduction

## What is SKlauncher

SKlauncher is a game launcher for the game Minecraft: Java Edition, with the simple goal to provide players an easy-to-use and
beautiful interface to manage the game and installed modifications, with an additional set of offline features.It provide`s the
gameplay that you want.It is specialy made for low end pc`s to make them run minecraft better.In sklauncher you can play with 
your cracked account(not official ) or premium account(official).
