# Introduction

## What is SKlauncher

SKlauncher is a game launcher for Minecraft: Java Edition. We have a simple goal to provide players with a beautiful yet easy-to-use
interface to manage the game as well as installing modifications with an additional set of offline features.

## Features

<ul className="emoji-list">
    <li>
        <span>☕</span> Built with Java 8, with support for Java 17-21
    </li>
    <li>
        <span>🔒</span> Supports authentication with Microsoft accounts
    </li>
    <li>
        <span>🚀</span> Frequently updated to provide new features
    </li>
    <li>
        <span>📦</span> Supports installing and managing modifications
    </li>
    <li>
        <span>💄</span> Designed with a clean approach in mind

</ul>
