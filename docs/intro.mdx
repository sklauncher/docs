# SKlauncher

**SKlauncher** is a game launcher for **Minecraft: Java Edition**, designed to provide players with an intuitive and visually appealing interface for managing the game, installed modifications, and various offline functionalities.

For {new Date().getFullYear() - 2014} years, SKlauncher has enabled users to log in using their Mojang accounts (prior to the 2023 migration to Microsoft) or play using offline accounts. Additionally, users have the option to create an SKlauncher account for enhanced features.
