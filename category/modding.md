# Modding

Minecraft: Java Edition can be modified in two main ways:

- **Native modifications**, such as **resource packs** and **data packs**, work in vanilla without extra tools.
- **External modifications** require a **mod loader** (Fabric, Forge, NeoForge, Quilt) to install and run.
