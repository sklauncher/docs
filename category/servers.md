# Servers

:::tip
Looking to **find** a server, not host one? There's no master server list, so you have to browse:

- [Xyrios](https://xyrios.com/minecraft/servers) covers Microsoft and Offline accounts (labelled Premium / Non-Premium respectively).
- [Official verified list](https://aka.ms/verifiedservers) is Microsoft only.
:::

There are two ways to host a server:

1. **Open a singleplayer world to LAN.** Easiest, but still an advanced topic.
2. **Run a dedicated server.**

> "Easiest" doesn't mean simple. Even with a clean setup, things can go wrong: firewall rules, ISP-level NAT, no real public IP, etc.
